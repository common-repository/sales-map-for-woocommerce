== Description ==

**Sales Map for WooCommerce**  is a plugin that shows sales on a google map. By installing this plugin, you can see which parts of the world/cities, regions you have sold products to. 

Sevengits Sales Map for WooCommerce Plugin will give you more insightful data about the places where most of the website orders came from. That too in a simplified look & feel.

Site owners can use these data in a Google Map which will show like points and it will be helpful to increase your brand value and trust among your visitors.

This is a great WooCommerce WordPress plugin to gain trust from your customers by showing that you have sold everywhere and even near their location. 

**Shortcode Available**

‘[sgitswcsm]’

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sales-map-woo.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[sgitswcsm]` in your pages/posts to show sales map.

== Frequently Asked Questions ==

= Does this need google map API? =

Yes. To work this plugin you need google map API.

= Activated Plugin but not showing any sales in map? =

The free version of this plugin doesn't show any historical sales. The plugin will show only orders that are placed after activating this plugin only.
